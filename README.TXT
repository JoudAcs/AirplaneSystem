# Airplane Ticket Reservation System

## Build Output Description

When you build a Java application project that has a main class, the IDE automatically copies all of the JAR files on the project's classpath to your project's `dist/lib` folder. 
The IDE also adds each of the JAR files to the `Class-Path` element in the application JAR file's `MANIFEST.MF`.
